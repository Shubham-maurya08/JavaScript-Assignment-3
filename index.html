Q1. What are the differences between var, let and const?

Ans: var:(Global Scope) It can be declare in a line and then initialized with a value in another lines and also declare and initialized in the same line. It can be redecalred but can be re-initialized and we can not assign variable same name in var keyword.
let:(Local Scope) It can be declare in a line and then initialized with a value in another lines and also declare and initialized in the same line. They can not be redecalred and but can be re-initialized. We can not have two variable with same name in let keyword.
Const:(Local Scope) They can be declared and initialized with a value in same line only. But can not be redecalred and cannot be re-initialized. We can not have two variable with same name in const keyword.

Q9. Explain the Pre-Increment and Post Increment in js.

Ans: Pre-increment (++i) - Before assigning the value to the variable, the value is incremented by one. 

            Ex :- let x = 3;
            let y = ++x;
            console.log(x) //it will give output 4 
            console.log(y) //it will give output 4.
    
    
            Post-increment (i++) - After assigning the value to the variable, the value is incremented. 
            Ex :- let x =3;
            let y = x++;
            console.log(x) //it will give output 4
            console.log(y) //it will give output 3
